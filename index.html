Upload an Excel file (.xlsx or .xls) to add multiple students at once.

üìã Excel Format Instructions

Your Excel file should have these columns (in order):

  1. Student Name - Full name (Required)
  2. Roll Number - Unique ID (Required)
  3. Class - e.g., "Class 10", "Class 11" (Required)
  4. Syllabus - "CBSE" or "Matric" (Optional)
  5. Phone Number - Contact number (Required)
  6. Monthly Fee - Amount in rupees (Required)
  7. Admission Date - Format: DD/MM/YYYY or YYYY-MM-DD (Optional)
  8. Status - "Active" or "Inactive" (Required)

Note: First row should be headers, data starts from row 2.
